# 3WaySurebet

A 3-way surebet calculator designed to help users convert freebet balances into withdrawable cash with maximum efficiency.

## Design Context

- **Register:** `product`
- **Aesthetic:** "The Cashier's Window" — Matter-of-fact, clear, and trustworthy. Avoids the aggressive, dark neon vibes of traditional sportsbooks.
- **Core Principles:** Input-first layout, confident numerical results, and high scannability.
